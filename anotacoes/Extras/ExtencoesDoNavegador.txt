Essas Extensões serviram tanto no Google Chrome como no Firefox.

1. JSON Viewer: serve para mostrar JSON bem mais formatado. 
Para instalar vá em: 
1.1 "mais ferramentas" no seu navegador.
1.2 Extensões.
1.3 Clique no menu "Extensões" no canto superior esquerdo.
1.4 Abrir loja web Chrome.
1.5 Pesquisar por JSON Viewer e instalar.

2. React Dev Tools: serve para mostrar os componentes React usados em sites de terceiros ao abrir o inspecionar elemento do seu navegador. Além de mostrar na sua aplicação React quais componentes estão demorando a carregar e outras vantagens no desenvolvimento de uma aplicação React.
2.1 "mais ferramentas" no seu navegador.
2.2 Extensões.
2.3 Clique no menu "Extensões" no canto superior esquerdo.
2.4 Abrir loja web Chrome.
2.5 Pesquisar por React Developer Tools e instalar.

3. Octotree: O Octotree nos permite navegar muito mais facilmente por repositórios no github, é uma otima ferramenta.
3.1 "mais ferramentas" no seu navegador.
3.2 Extensões.
3.3 Clique no menu "Extensões" no canto superior esquerdo.
3.4 Abrir loja web Chrome.
3.5 Pesquisar por Octotree e instalar.
3.6 Veja que ao entrar em algum repositório do GitHub vai ter uma janelinha com o nome Octotree na parte lateral esquerda.
